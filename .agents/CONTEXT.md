# Instrucciones globales

## Idioma
Responde siempre en español.

## Stack tecnológico
- Backend: Django/Python
- Frontend: React
- Base de datos: PostgreSQL

## Estilo de código
- Usa type hints en Python
- Componentes React funcionales con hooks
- Variables y funciones en snake_case (Python) / camelCase (JS)

## Preferencias generales
- Crea el plan de trabajo en la carpeta .agent.
- Explica los cambios importantes antes de aplicarlos
- Sugiere mejoras de seguridad cuando las detectes
- Contexto: sistemas de nómina con regulaciones chilenas (AFP, vacaciones, etc.), el sistema esta en constante cambio y evolución, por lo que debes estar atento a los cambios y adaptarte a ellos.
- El sistema es multi pais, por lo que debes tener en cuenta las regulaciones de cada pais.

# Reglas
- En las implementaciones considera buenas practicas de codigo, pero estas intervensiones deben ser quirurgicas, es decir, solo modificar lo estrictamente necesario para cumplir con el objetivo.
